# config.nvim

Personal Neovim configuration

## Directories

### Windows

Config: `~/AppData/Local/nvim`
Data: `~/AppData/Local/nvim-data` 

### Mac OS

Config: `~/.config/nvim`
Data: `~/.local/share/nvim`

## Features

- lazy package manager
- mini.nvim status line
- Tokionight theme
- basic lua settings
